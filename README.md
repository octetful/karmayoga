# Karmayoga

Java library for Task Organization.

[![CircleCI](https://circleci.com/gh/sighosh2/karmayoga.svg?style=svg)](https://circleci.com/gh/sighosh2/karmayoga)

## Tech Stack
* Java 11
* Gradle
* JUnit 5
* Mockito


## Commands:

* To build the project use:

```bash
./gradlew build
```

* To run all the tests use:
```bash
./gradlew test
```
